# Skaner wyrażeń matematycznych

## Zbiór tokenów

| Kod             | Opis                                                               |
| --------------- | ------------------------------------------------------------------ |
| `LICZBA`        | Liczba całkowita — ciąg cyfr dziesiętnych                          |
| `IDENTYFIKATOR` | Identyfikator — litera, po której mogą następować litery lub cyfry |
| `PLUS`          | Operator dodawania                                                 |
| `MINUS`         | Operator odejmowania                                               |
| `RAZY`          | Operator mnożenia                                                  |
| `DZIELENIE`     | Operator dzielenia                                                 |
| `NAWIAS_L`      | Nawias otwierający                                                 |
| `NAWIAS_P`      | Nawias zamykający                                                  |
| `KONIEC`        | Koniec wyrażenia                                                   |
